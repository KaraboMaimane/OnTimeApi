## Getting Started

To get started with the OnTimeApi project, follow these steps:

1. **Clone the repository**:
   ```sh
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```sh
   cd OnTimeApi/OnTimeApi
   ```

3. **Build the project**:
   ```sh
   dotnet build
   ```

4. **Run the project**:
   ```sh
   dotnet run
   ```
